VidaOS
=========================
A tiny RTOS for ARM Cortex-M3/M4 with basic functionï¼š

* Tasks scheduled by priority and time slice
* Event
* Semaphore
* Mailbox
* Memory Block
* Flag Group
* Mutex
* Quick rollback
* Timer
* CPU usage and free stack measure


Software Environment
------------

* MDK Version 5.20.0.0


Hardware
------------
* STM32F429 MCU Discovery Kit
